## Guide

### 1. Installing the macro
To use the macro, extract the folder and double-click `START.bat` to run the macro. There might be a warning saying "unknown file" or it might block opening. This is expected, as batch files downloaded from the internet are normally flagged by Windows Defender. Just click "More info" and "Run anyway".

### 2. Configuring/Running the macro
After you have the macro open, configure your settings, and press the Start button (default `F1`) to run it. 
There is an extensive guide in our Discord server explaining each feature and setting to help you configure it to your exact needs.

## Keybinds (Default)
* **`F1`** - Starts the macro. If you click it while already active, nothing happens.
* **`F2`** - Pauses the macro (only works when the macro is actively running).
* **`F3`** - Reloads the macro. To fully close the macro, click the "X" button at the top right of the GUI.
